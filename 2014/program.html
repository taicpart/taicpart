Program

Date and Time: April 4, 2014 09:30-17:30

Location: Hyatt Regency Arcade, downtown Cleveland

09:00 - 10:30: Welcome and Keynote

  • 09:00, Welcome: Michael Felderer and Gregory M. Kapfhammer
  • 09:30, Keynote: Dr. Simon Poulding: Aligning the Theory and Practice of Software Testing

10:30 - 11:00: Coffee Break

11:00 - 12:30: Improving Automation, GUI models and Testability

  • 11:00, Test Automation Improvement Model (TAIM), Sigrid Eldh, Andreas Ermedahl, Kenneth Andersson and Kristian Wiklund
  • 11:30, Murphy Tools: Utilizing Extracted GUI Models for Industrial Software Testing, Pekka Aho, Matias Suarez, Teemu Kanstren and Atif Memon
  • 12:00, On an Embedded Software Design Architecture for Improving the Testability of In-Vehicle Multimedia Software, Brownson Obaridoa Obele and Dongyul Kim

12:30 - 14:00: Lunch Break

14:00 - 15:30: A panel of industrialists and academics discussing the challenges in knowledge transfer and industry/academia collaborations

15:30 - 16:00: Coffee Break

16:00 - 17:00: Test Requirements, Environment, Strategy and Design

  • 16:00, Viewpoint-based Test Requirement Analysis Modeling, Yasuharu Nishi
  • 16:30, Practical Challenges in Test Environment Management, Rudolf Ramler and Johannes Gmeiner
  • 16:50, Trial of Organizing Software Test Strategy via Software Test Perspectives, Shinji Kawaguchi

17:00 - 17:30: Discussion and Closing
